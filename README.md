Adventure Framework
===================

The AdventureFramework project exists to provide an easy way to create and configure complex rulesets in Minecraft.
 - Ability System
 - Protection (WorldGuard)
 - Automatic backup management
 - Multiworld support(in bukkit aka. multiverse)
 - Deathchests
 - Huge collection of utility commands
 - Remote server access and management with a versatile API
 - And many more!!!
